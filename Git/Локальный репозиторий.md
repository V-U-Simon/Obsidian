## Репозиторий

```shell
# cоздание нового репозитория
git init
# связывание его с удаленным хранилищем
git remote add name_repo www@85.143.172.241:/home/www/code/project_dir
```

```shell
# Клонирование существующего удаленного репозитория на локальную машину
```

## Ветки

```shell
# Создание
git branch testing      # создать ветку
git checkout -b testing # создать и переключится

# Переключиться
git checkout testing    # переключится на другую ветку

# Просмотр локальных веток
git branch
```
